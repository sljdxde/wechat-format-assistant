# 微信排版助手 ✍️

> 将 Markdown 文档一键转换为微信公众号精美排版，支持多种主题，复制即可粘贴。

## 🌐 在线使用

部署到 GitHub Pages 后，直接在浏览器中访问即可使用。

本地使用：直接双击打开 `index.html` 即可。

## ✨ 功能特性

- **实时预览**：左侧输入 Markdown，右侧即时显示排版效果
- **4 种精美主题**：简约 / 优雅 / 科技 / 清新
- **丰富的 Markdown 支持**：标题、加粗、斜体、代码块、表格、引用、列表、链接、图片、分隔线
- **一键复制**：复制为富文本，直接粘贴到微信公众号编辑器
- **文件上传**：支持拖拽 `.md` 文件或点击上传
- **面板可拖拽**：自由调整编辑器和预览区的宽度比例
- **零依赖**：纯 HTML/CSS/JS，无需构建工具

## 📁 项目结构

```
├── index.html          # 主页面
├── css/style.css       # 样式
├── js/
│   ├── themes.js       # 主题定义（4 种主题）
│   ├── converter.js    # Markdown → HTML 转换器
│   └── app.js          # 应用逻辑
├── project/            # 原始飞书 Block 插件代码（备份）
└── README.md
```

## 🚀 部署到 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` 分支，目录选 `/ (root)`
4. 保存后即可通过 `https://<username>.github.io/<repo>/` 访问
